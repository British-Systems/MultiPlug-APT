Debian Linux Installation

[Option 1] Install with Repository Updates

A Linux package repository is available for Debian-based Linux versions, this includes Raspberry Pi OS.

  1. To add MultiPlug Debian Repository to your instance of Linux, Add the following line to /etc/apt/sources.list (Only need to do this once)
  2. deb [trusted=yes] https://apt.multiplug.app ./
  3. Run the following command:
  4. sudo apt update
  5. To install MultiPlug run the following command:
  6. sudo apt-get install multiplug
  7. Multiplug will boot and on system startup, to access it use a web browser and IP address of the device

Note: The Mono .Net Framework needs to be installed using: sudo apt-get install mono-complete

To stop Multiplug use the command: systemctl stop multiplug

To start MultiPlug use the command: systemctl start multiplug

[Option 2] One Time Install

Manually download the .deb file and install once.

  1. Check for the the latest .deb file version here.-2.2.67.deb
  2. curl –O https://apt.multiplug.app/multiplug-2.2.68.deb>
  3. sudo apt install ./multiplug-2.2.68.deb
  4. Multiplug will boot and on system startup, to access it use a web browser and IP address of the device

Note: The Mono .Net Framework needs to be installed using: sudo apt-get install mono-complete

To stop Multiplug use the command: systemctl stop multiplug

To start MultiPlug use the command: systemctl start multiplug
